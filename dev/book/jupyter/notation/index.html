Notation

Accents¶

  • x\mathbf{x}x (bold, non-italic typeface) Matrix or vector

  • xxx (non-bold, italic typeface) Scalar or array

Variables¶

  • nfn_fnf​ Number of frequency lines

  • ncn_cnc​ Number of control channels

  • non_ono​ Number of output signals

  • nsn_sns​ Number of samples in a signal

  • nosn_{os}nos​ Number of samples in an oversampled output signal

  • Gvv\mathbf{G}_{vv}Gvv​ CPSD matrix for the output signals

  • Gxx\mathbf{G}_{xx}Gxx​ CPSD matrix for the responses (i.e. the specification)

  • Hxv\mathbf{H}_{xv}Hxv​ Transfer function matrix between the responses and the output signals

  • X\mathbf{X}X Response spectra (FFT)

  • V\mathbf{V}V Output spectra (FFT)

  • T\mathbf{T}T Transformation matrix

  • A\mathbf{A}A State or system matrix for state space equations of motion

  • B\mathbf{B}B Input matrix for state space equations of motion

  • C\mathbf{C}C Output matrix for state space equations of motion

  • D\mathbf{D}D Feedthrough or feedforward matrix for state space equations of motion

  • x\mathbf{x}x State vector for state space equations of motion

  • y\mathbf{y}y Output vector for state space equations of motion

  • u\mathbf{u}u Input vector for state space equations of motion

  • M\mathbf{M}M Mass matrix for 2nd-order differential equations of motion

  • C\mathbf{C}C Damping matrix for 2nd-order differential equations of motion

  • K\mathbf{K}K Stiffness matrix for 2nd-order differential equations of motion

Operations¶

  • ⬚+^++ Pseudoinverse of a matrix

  • ⬚H^HH Complex-conjugate transpose of a matrix (hermitian)
