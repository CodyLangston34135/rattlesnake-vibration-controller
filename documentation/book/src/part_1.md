This Part provides a general overview of Rattlesnake software.  @sec:acquiring_and_running_rattlesnake provides an overview of how to acquire and run the software.  @sec:using_rattlesnake describes the general software workflow.

This Part with only deal with "global" Rattlesnake parameters. For hardware-specific parameters, see @sec:rattlesnake_hardware.  For environment-specific parameters, see @sec:rattlesnake_environments.
